Webscraper to scrape grades from Studentweb.

Copy-paste the 'example.env' to 'credentials.env'.
Replace the example text in the .env file with:
- Student ID
- Student Code
- Ntfy custom url
- Path for where you want the results.json to be saved

Run the 'run.sh' to scrape.
Additionally, it's possible to setup a cronjob to make this run every once in a while.
Recommendations: Not more often than once an hour.
